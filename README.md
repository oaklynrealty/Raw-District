# Raw District Ads Campaign

Dedicated paid-traffic landing page for Raw District by IMTIAZ.

## Build

```bash
node scripts/build.mjs
node scripts/check.mjs
```

The generated site is written to `dist/`.

## Variants

- Variant A: `/` or `/raw-district/`, with step 1 of the lead form directly in the hero.
- Variant B: `/variant-b/` or `?variant=b`, with a CTA-first hero and short form inside a modal or mobile bottom sheet.

Both variants use the same offer and CTA wording.
